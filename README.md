# Elite Financial - Site Web

Une plateforme moderne et sécurisée de gestion d'investissements avec design premium fintech.

## 📁 Structure du Projet

```
site-web/
├── index.html          # Page d'accueil
├── login.html          # Page de connexion
├── register.html       # Page d'inscription
├── features.html       # Page des fonctionnalités
├── about.html          # Page à propos
├── css/
│   └── styles.css      # Feuille de styles principale
├── js/
│   └── script.js       # JavaScript principal
└── README.md           # Ce fichier
```

## 🎨 Design

- **Police**: Inter (principale) et JetBrains Mono (monospace)
- **Icônes**: Font Awesome 6.4.0
- **Couleurs**: 
  - Primary: #3b82f6 (Bleu Fintech)
  - Accent: #10b981 (Vert)
  - Dark: #030712 (Noir profond)

## ✨ Fonctionnalités

- Design responsive (mobile-first)
- Navigation fluide
- Animations CSS
- Formulaires interactifs
- Support multi-langue (FR/EN)
- Thème sombre professionnel

## 🚀 Installation

1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
3. Aucune installation supplémentaire requise

## 📱 Pages

- **Accueil** (`index.html`) - Page d'accueil avec hero section
- **Connexion** (`login.html`) - Formulaire de connexion
- **Inscription** (`register.html`) - Formulaire d'inscription
- **Fonctionnalités** (`features.html`) - Présentation détaillée
- **À propos** (`about.html`) - Information sur l'entreprise

## 🔐 Sécurité

Cette plateforme est une simulation éducative. Aucune transaction réelle n'est effectuée.
